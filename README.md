# SocialGenius

SocialGenius is a modern, AI-powered social media management application.
This application allows you to schedule posts, manage multiple social media accounts, and analyze engagement seamlessly.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the development server:
   `npm run dev`
3. Open `http://localhost:3000` in your browser.
